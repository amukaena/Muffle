# Muffle - 화면 설계

## 화면 구성
단일 화면(Single Screen) 구조. 수면 전 최소 조작을 위해 모든 기능을 한 화면에 배치.

## 레이아웃

```
┌─────────────────────────────┐
│          Muffle              │  ← 앱 이름 (상단)
│                              │
│                              │
│      ┌───────────────┐       │
│      │  종료 시각      │       │
│      │   07:00        │       │  ← 탭하여 시각 변경
│      │                │       │
│      │  남은 시간      │       │
│      │   6시간 32분    │       │  ← 재생 중일 때만 표시
│      └───────────────┘       │
│                              │
│                              │
│          advancement          │
│      ┌──────────────┐        │
│      │   ▶ / ■      │        │  ← 재생/정지 토글 버튼
│      └──────────────┘        │
│                              │
│   🔈 ━━━━━━━━●━━━━━ 🔊      │  ← 볼륨 슬라이더
│                              │
└─────────────────────────────┘
```

## UI 요소 상세

### 1. 종료 시각 표시
- 기본값: 다음 날 오전 7:00
- 탭하면 TimePicker 다이얼로그 표시
- 24시간 또는 12시간(AM/PM) 형식 (시스템 설정 따름)

### 2. 남은 시간 표시
- 재생 중일 때만 표시
- "N시간 N분" 형식
- 1분 간격 갱신 (배터리 절약)

### 3. 재생/정지 버튼
- 정지 상태: ▶ (재생 아이콘)
- 재생 상태: ■ (정지 아이콘)
- 크게 배치하여 어두운 환경에서 쉽게 탭 가능

### 4. 볼륨 슬라이더
- 앱 내부 볼륨 제어 (0~100%)
- 기본값: 70%
- 시스템 미디어 볼륨과 독립

## 테마
- **다크 테마 고정** (시스템 설정 무시)
- 배경: 어두운 색 (#121212 계열)
- 강조색: 차분한 파란색 계열 (수면 분위기)
- 텍스트: 낮은 밝기의 흰색 (눈부심 방지)

## Foreground Service 알림
재생 중 상태바 알림 표시:
- 제목: "Muffle 재생 중"
- 내용: "07:00에 종료 예정"
- 액션 버튼: 정지
