## 의존성 (최소화)

| 라이브러리 | 용도 |
|-----------|------|
| Jetpack Compose BOM | UI |
| Material3 | 디자인 컴포넌트 |
| Lifecycle ViewModel | 상태 관리 |

별도 오디오 라이브러리 불필요 - Android 기본 AudioTrack API 사용.
DI 프레임워크 불필요 - 단일 화면 앱이므로 수동 의존성 주입으로 충분.
