# Muffle - 프로젝트 개요

## 앱 소개
수면 시 옆집 소음을 덮기 위한 브라운 노이즈 재생 앱.
지정한 시각까지 브라운 노이즈를 무한 반복 재생하고 자동 종료한다.

## 핵심 기능
| 기능 | 설명 |
|------|------|
| 브라운 노이즈 재생 | 재생/정지 버튼으로 제어 |
| 종료 시각 설정 | 특정 시각(예: 오전 7:00) 지정 시 자동 종료 |
| 볼륨 조절 | 앱 내 볼륨 슬라이더 |
| 남은 시간 표시 | 종료 시각까지 남은 시간 표시 |
| 백그라운드 재생 | 화면 꺼진 상태에서도 재생 유지 |

## 기술 스택
| 항목 | 선택 |
|------|------|
| 언어 | Kotlin |
| UI | Jetpack Compose |
| 최소 SDK | 26 (Android 8.0) |
| 타겟 SDK | 34 (Android 14) |
| 아키텍처 | MVVM |
| 백그라운드 재생 | Foreground Service |
| 오디오 | AudioTrack (PCM 기반 브라운 노이즈 실시간 생성) |
| 테마 | 다크 테마 기본 |

## 설계 원칙
- **심플한 UI** - 한 화면에 모든 기능 배치, 수면 전 최소 조작
- **낮은 배터리 소모** - 오디오 파일 대신 PCM 실시간 생성으로 디스크/메모리 I/O 최소화
- **안정적 백그라운드** - Foreground Service + 알림으로 시스템에 의한 강제 종료 방지

## 배터리 최적화 전략
- AudioTrack으로 브라운 노이즈를 실시간 생성 (별도 오디오 파일 불필요)
- 화면 꺼진 상태에서 UI 갱신 중지
- WakeLock 최소 사용 (AudioTrack 재생 중에는 시스템이 자동으로 CPU 유지)
- 타이머 종료 시 모든 리소스 즉시 해제
